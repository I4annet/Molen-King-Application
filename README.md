## 📖 About

Molen King Application is a modern cashier and business management system developed using **Flutter** and **Supabase**.

The application is designed to simplify daily business operations, including cashier transactions, inventory management, employee attendance, expense tracking, and sales reporting—all within a single application.

This project implements **Role-Based Access Control (RBAC)** by separating access between **Admin/Owner** and **Cashier**.

---

## ✨ Features

### 👑 Admin

- Dashboard Overview
- Sales Reports
- Employee Attendance Monitoring
- Stock Management
- Revenue & Profit Analysis

### 🧾 Cashier

- Cashier Transaction System
- Daily Attendance (Check-In / Check-Out)
- Expense Recording
- Stock Monitoring

### 🔐 Authentication

- Login
- Register
- Role-Based Access (Admin & Cashier)
- Secure Authentication using Supabase Auth
